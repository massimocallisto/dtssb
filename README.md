# Workshop on Digital Technologies for Smart and Sustainable Buildings (DTSSB) Website

This repository contains the static website for the DTSSB workshop

## Purpose

The project is used to publish and maintain the workshop website, including:

- the workshop overview and scope
- topics of interest
- submission information
- organizers and program committee
- workshop history and contact details

## Notes

- This is a static site, so no build step is required for basic edits.
- Update the HTML, CSS, and assets directly when preparing a new edition of the workshop.
